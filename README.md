# Imperium Advisor

Интерактивный MVP стратегического кабинета римского императора по требованиям `pdr.md` и `prd_addendum.md`.

## Запуск

Откройте `index.html` в современном браузере. Сборка и сервер не требуются.

## Реализовано

- масштабирование и перемещение карты;
- шесть режимов отображения;
- карточки городов и легионов;
- подробная карта с названиями 42 провинций эпохи Траяна;
- постоянный советник с сохранением истории переписки в браузере;
- 13 пограничных легионов: Британия, Рейн, Дунай и Сирия;
- содержательные слои населения, экономики, торговли, войск и угроз;
- верхняя панель показателей и управление годом;
- интерактивный локальный советник;
- разделы управления империей;
- адаптивная компоновка.

Сгенерированная карта хранится локально в `assets/roman-map.png`.
